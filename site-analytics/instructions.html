Uputstvo za implementaciju analitike

Korak 1: Dobijanje kredencijala

  1. Idite na: http://localhost/PORTFOLIO/site-analytics/clients.php
  2. Kliknite "Add Client" i popunite formu
  3. Zapišite dobijeni API ključ
  4. Idite na: http://localhost/PORTFOLIO/site-analytics/websites.php
  5. Dodajte svoj sajt i zapišite dobijeni Website ID

Korak 2: Dodavanje koda za praćenje

Dodajte sledeći kod u HEAD sekciju vašeg sajta:

<!-- Analytics Tracking Code -->
<script>
window.analyticsConfig = {
    websiteId: 'YOUR_WEBSITE_ID',  // Zamenite sa vašim Website ID
    apiKey: 'YOUR_API_KEY'         // Zamenite sa vašim API ključem
};
</script>
<script src="http://localhost/PORTFOLIO/site-analytics/tracker.js"></script>

Korak 3: Test implementacije

Proverite da li tracking radi:

  1. Otvorite svoj sajt u browseru
  2. Pritisnite F12 za Developer Tools
  3. U Network tabu proverite da li se poziva track.php
  4. Posetite: http://localhost/PORTFOLIO/site-analytics/view_stats.php?id=YOUR_WEBSITE_ID

Primer kompletne stranice

<!DOCTYPE html>
<html>
<head>
    <title>Moj Sajt</title>
    
    <!-- Analytics Tracking -->
    <script>
    window.analyticsConfig = {
        websiteId: '1',        // Vaš Website ID
        apiKey: 'test123api'   // Vaš API ključ
    };
    </script>
    <script src="http://localhost/PORTFOLIO/site-analytics/tracker.js"></script>
</head>
<body>
    <h1>Dobrodošli na moj sajt</h1>
    <!-- Vaš sadržaj -->
</body>
</html>

Provera statistike

  • Dashboard: http://localhost/PORTFOLIO/site-analytics/dashboard.php
  • Detaljna statistika: http://localhost/PORTFOLIO/site-analytics/view_stats.php?id=YOUR_WEBSITE_ID
  • Izveštaji: http://localhost/PORTFOLIO/site-analytics/reports.php
